# Pre-commit Setup

Pre-commit hooks đã được cài đặt thành công!

## Features

- ✅ Black (Python formatter)
- ✅ isort (Import sorter)
- ✅ Flake8 (Linter)
- ✅ Ruff (Fast linter)
- ✅ MyPy (Type checker)
- ✅ CSS Validator (cross-platform Python script)

## Usage

Pre-commit sẽ tự động chạy khi bạn commit. Nếu có lỗi, commit sẽ bị chặn.

Để chạy thủ công:

```bash
pre-commit run --all-files
```
